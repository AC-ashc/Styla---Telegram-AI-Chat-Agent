# Styla - Telegram AI Chat agent

## Overview
Styla is an AI-powered Telegram customer support agent for Tee Bazaar Clothing Store. It assists customers by answering inquiries related to products, product care, shipping, refunds, returns, and general store information. Powered by an AI knowledge base, Styla provides accurate and instant responses to customer questions. It can also send requested information directly to customers via email, ensuring a seamless and efficient support experience.

## Workflow

Telegram message trigger → AI agent → Gemini Chat model → Google Sheet → Gmail → Telegram Auto Response

## Tech Stack

* n8n
* Gemini model API
* AI agent
* Telegram chat trigger
* Google sheet 
* Gmail
* API calls
* JSON
* OAuth
* Webhook
